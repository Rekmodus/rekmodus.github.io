First Person Horror Game

Collaborated with a team of 3 to develop a 3d horror puzzle game built using Javascript, HTML, GLSL, WebGL, and Blender. It was made for my computer graphics final project. My job was to build the core features and come up with overall idea.
Features:
- Shadows. 2 pass z buffer. Creates dynamic light depth texture using projection from light
- Collision: AABB(Axis-Aligned Bounding Box)
- First person controller. Used various matrix transformations to reproduce classic controls
- Bump mapping. Faked by adding one line of code to the shader
- Flashlight. Made circular based on distance from screen center, combine with shadow shader
- Fog - based on distance from camera, exponential fog factor, mix original with black by fog factor
- Physics based simulation
- Text system - 2d planes in 3d space

FPGA-Based Piano Synthesizer

Implemented an FPGA-based Piano Synthesizer using the Basys 3 development board using Verilog and Vivado. It was the final project for Computer Systems Architecture.
- Utilize the 15 switches on the Basys 3 to represent musical notes.
- Implement a special switch (rightmost switch) to toggle between major and minor scales.
- Implement 5 buttons to reroute the switch notes to different corresponding scales.
- Use the 7-segment display to show the current key being played.
- Generate audio by sending serial data (MIDI Signals) to a Python script that figures out scale, major, and notes from the 8 bits.
